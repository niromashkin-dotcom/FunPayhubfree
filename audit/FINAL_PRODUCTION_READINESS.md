# Final Production Readiness Audit

Этот документ подводит итог второго этапа "Production Hardening" перед первым запуском на VPS. 
Все обнаруженные риски, блокеры и потенциальные утечки устранены. 

## Что готово к бою (Ready for Production)
- **Ядро**: `EventBus`, `Orders`, `Ledger`, `CCE` (коммуникация с клиентом) полностью функциональны и связаны в единый пайплайн.
- **Безопасность (Idempotency)**: Двойная оплата исключена. Добавлена защита от повторной отправки вебхуков. Прибыль и товары выдаются строго один раз.
- **Отказоустойчивость**: Внедрен `EventJournal`. При падении сервера, события восстанавливаются и обрабатываются заново.
- **Состояние ожидающих заказов (Persistence)**: Оба трекера (`order_tracker.py` и `autodonate_plugin.py`) теперь сохраняют свои данные о висящих заказах в `data/state/pending_orders.json` и `data/autodonate/autodonate_pending_orders.json`. Рестарт VPS больше не приведет к потере денег или зависанию заказа.
- **Git история**: Проверена и очищена. Личные `chat_id` и токены FunPay не утекли в публичную историю, а все генеративные `.json` файлы с пользовательскими данными внесены в `.gitignore`.
- **Клиентская часть**: Тексты автоответов откорректированы (убрано ложное обещание возврата за 24 часа, добавлены корректные русскоязычные уведомления).

## Что отключено (Disabled)
Следующие плагины и функции переведены в состояние `enabled: false`, чтобы избежать ложных срабатываний и холостых заказов. Их следует включать только после настройки API ключей:
- **ShopClaude** (AutoDonate)
- **Kosell** (AutoDonate)
- **AI Team** (авто-управление проектом)
- **AutoDelivery** (общая автовыдача)

## Какие переменные окружения нужны (Environment Variables)
Для корректной работы на VPS (в файле `.env`) необходимо прописать:
- `FUNPAY_API_TOKEN` - токен авторизации FunPay (если используется прямое подключение)
- `TELEGRAM_BOT_TOKEN` - токен Telegram-бота для логов и управления
- `TELEGRAM_ADMIN_CHAT_ID` - ID чата для уведомлений об ошибках
- `FUNPAYHUB_APP_URL` - публичный адрес хаба (например, `http://your-vps-ip:5000`)
- *(Опционально)* API ключи для `GorgonaBoosts` и `HoldBoost` (или любых других включенных поставщиков).
