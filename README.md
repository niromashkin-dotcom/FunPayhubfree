# FunPayHub

**FunPayHub** — это продвинутый комплекс для полной автоматизации работы продавца на площадке FunPay.ru.  
Проект вырос из старых решений и превратился в микросервисную архитектуру с собственным API, плагинами и удобным ботом управления в Telegram.

## 🚀 Основные возможности

* **Полная автоматизация продаж:** Выдача цифровых товаров, услуг, игровой валюты и подписок 24/7.
* **Интеграция с поставщиками (AutoSMM):** Автоматическая закупка и исполнение заказов через GorgonaBoosts, HoldBoost, Kosell и других.
* **Автоподнятие предложений (AutoBump):** Умное поднятие лотов для удержания верхних позиций в выдаче.
* **Мониторинг конкурентов:** Анализ цен и рынка для формирования лучшего предложения.
* **Модульная система (Плагины):** Легко подключаемые модули для расширения функционала (AutoDonate, AutoReply, и т.д.).
* **Telegram Control Bot:** Удобная панель управления прямо в Telegram. Проверяйте баланс, статус лотов и состояние системы с телефона.

## 🛠 Архитектура

FunPayHub состоит из двух основных компонентов, которые могут работать как локально, так и разворачиваться на облачных платформах (например, Render.com):

1. **Core Hub (Фоновый процесс):** Отвечает за коммуникацию с API FunPay, запуск плагинов, мониторинг и предоставляет внутренний REST API.
2. **Telegram Bot:** Подключается к Core Hub по API и предоставляет интерфейс взаимодействия для владельца.

## ⚙️ Технологии

* **Язык:** Python 3.10+
* **Фреймворки:** Flask (для внутреннего API), pyTelegramBotAPI (для бота)
* **Архитектура:** EventBus, Singleton Manager, Плагины
* **Деплой:** Поддерживается облачный деплой (render.yaml, Procfile, ENV-секреты).

## 🔒 Безопасность

* Авторизация Telegram-бота на базе `chat_id` и системы паролей.
* Хранение ключей (`golden_key`, API токенов) через безопасные Environment Variables, исключая утечки через исходный код.
